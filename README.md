1. Clone the repository:
   ```bash
   git clone https://github.com/YatharthChauhan2362/prod-nensi-portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd prod-nensi-portfolio
   ```

4. Start the development server:
